# Criação de Relatório Dinâmico - Desafio de Projeto DIO

Este projeto consiste na criação de um relatório dinâmico utilizando o Power BI, como parte de um desafio da DIO. O objetivo é proporcionar uma análise interativa e adaptável dos dados, permitindo que o usuário explore as informações de diferentes maneiras, facilitando a tomada de decisões baseada em dados.
